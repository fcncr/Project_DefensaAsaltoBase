Instrucciones de ejecución

Este proyecto fue desarrollado en Python utilizando Tkinter para la interfaz gráfica.

Requisitos

Antes de ejecutar el programa, asegúrese de tener instalado:
Python 3.
Tkinter.
Todos los archivos del proyecto en la misma carpeta.
En Windows, Tkinter normalmente viene incluido con Python.

Archivo principal
El archivo principal del proyecto es:
main.py
El programa debe ejecutarse desde este archivo.


Cómo ejecutar el proyecto:


Opción 1: Desde terminal

Abrir la terminal dentro de la carpeta del proyecto y ejecutar:
python main.py
Si el comando anterior no funciona, usar:
py main.py


Opción 2: Desde Visual Studio Code

Abrir la carpeta completa del proyecto en Visual Studio Code.
Abrir el archivo main.py.
Ejecutar el archivo desde el botón de Python o desde la terminal con:
python main.py
Opción 3: Desde IDLE
Abrir IDLE.
Abrir el archivo main.py.
Presionar F5 para ejecutar el programa.
Archivos necesarios

Para que el programa funcione correctamente, se debe mantener la carpeta completa del proyecto. No se recomienda ejecutar únicamente el archivo main.py separado de los demás archivos.

El proyecto puede incluir archivos para:

Ejecutar el programa.
Usuarios registrados.
Contraseñas.
Victorias.
Ranking.
Documentación.


Si el programa no abre, ejecutar desde terminal para ver el error.
